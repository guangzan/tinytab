## Install

1. Clone source code.
2. Open your Edge or Chrome.
3. Open extensions page.
4. Open developer mode.
5. Load unpacked.
6. Select dist directory.

> You can also search for "TinyTab" in Microsoft Store.

## Run and build

* `npm install`
* `npm start`
* `npm run build`
